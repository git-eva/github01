学习Git如何使用

常用指令

  • git init
  • git config
  • git config -l
  • git status
  • git branch -av
  • git clone 远端地址
  • git add .
  • git commit -m'提交内容'
  • git push
  • git pull
  • git fetch
  • git remote add 远端库名称 远端库地址
  • git pull origin master --allow-unrelated-histories
  • git checkout
  • git branch 分支名字
  • git branch -av 查询所有分支
  • git merge 合并分支
  • git merge develop

基本术语

  1. 仓库
  2. 指令
